# Security Policy

## Scope

Korzent is a deterministic execution governance standard.

Security issues relevant to:

- Canonicalization correctness
- Signature verification
- Schema validation
- Hash derivation
- Receipt integrity

should be reported privately.

## Reporting

Please open a private security advisory or contact the maintainer directly.

Do not publish proof-of-concept exploits publicly before disclosure is acknowledged.

## Notes

- Example keys in this repository are demo-only.
- No production secrets are stored in this repository.
